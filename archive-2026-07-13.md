## M1 — реальный инструмент (2026-07-11 → 2026-07-12): слайсы 0-5 СДАНЫ

**Спека M1** `docs/superpowers/specs/2026-07-11-synapse-m1-tool-design.md` — «кодить в машине голосом, дирижируя настоящей Корой (Claude Code) на Маке». 3 раунда критики (75 находок, `a0f7536`), скоуп срезан v1-ядро vs M1.1, DoD-гейты §6.

- **Слайс 0 — хост-синглтон** `c0c856b` (94 теста): `build_host` (долгоживущий стейт) + `build_session_pipeline` (per-connection адаптеры). Живой стейт больше не пересоздаётся на каждый WebRTC-offer.
- **Слайс 1 — настоящая Кора** `132215f`+`fca6053` (123 теста, live-доказан): `bridge/kora.py::KoraRunner` через `claude-agent-sdk`, свежий клиент на задачу в workspace, allowlist-маппер событий, структурный анти-зомби (try/finally + watchdog + synthetic fail). Уроки: cwd НЕ песочница — граница = permission-гейт; `allowed_tools=[]`+`setting_sources=[]` обязательны; system_prompt называет workspace.
- **Слайс 2 — push-канал** `06ac95f` (132 теста): completion-SPEAK Коры слышен в 🔇 — хост инжектит `TTSSpeakFrame` в живой PipelineTask через `queue_frame` (переиспользует арбитр+TTS). Гвард `has_finished()` от тихого дропа, bind внутри lock.
- **Слайс 3 — answer_kora (E5)** `d11f77a` (155 тестов, live-доказан off-menu nonce'ом): Кора зовёт AskUserQuestion → гейт паркует на `asyncio.Future` → вопрос озвучен → 5-й инструмент `answer_kora` резолвит → Кора продолжает ТУ ЖЕ задачу. CLI не валидирует answer∈labels → свободный ответ = тот же механизм.
- **Слайс 4 — security-гейт** `a8dd919` (173 теста, live-DoD «прочитай .env» → deny): КЛЮЧЕВОЕ — `can_use_tool` НЕ универсален (мимо Read/Glob/Bash), весь гейт перенесён на `PreToolUse` hook. Secret-path denial (casefold), Bash/egress DENY (v1), NO-EXFIL backstop (SPEAK-темплейт из task_text).
- **Слайс 5 — PWA + реконнект** `aaac2ce` код + `be85dfa` спека (252 теста): транспорт = Tailscale (`https://temirlans-macbook-air.tail97957e.ts.net` → :7860, tailnet-only), PWA-обёртка над prebuilt (инжект в `</head>` с RuntimeError-анкером, роуты ДО mount), reconnect.js БЕЗ wall-clock (iOS морозит таймеры) — правда состояния через `GET /client/session-alive`, приветствие «С возвращением…» под латчем, replay недоставленных critical через `unspoken(min_age_s=5)`.

**Репо на GitHub (2026-07-12):** `github.com/Terobyte/agentx`, PRIVATE, `.env` гитигнорен.
