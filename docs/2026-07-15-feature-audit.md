## 1. Шипнуто

### Голосовое ядро (M0 + M1, слайсы 0–5)

- **Голосовой контур end-to-end**: микрофон → WebRTC с AEC → wake/арбитр → **настоящая Кора**
  через `claude-agent-sdk` (`synapse/bridge/kora.py::KoraRunner`) — реальная задача создаёт файл
  на диске в workspace, live-доказано. Анти-зомби структурный.
- **Push-канал**: completion-SPEAK Коры слышен голосом даже в муте — host инжектит
  `TTSSpeakFrame` в живой PipelineTask (`queue_frame`), переиспользуя арбитр+TTS.
- **`answer_kora`**: Кора упирается в вопрос → задаёт его голосом → ответ пользователя
  (голосом или текстом) продолжает **ту же** задачу через `asyncio.Future`. Free-form ответы
  мимо меню принимаются (CLI `AskUserQuestion` не валидирует answer∈labels — проверено probe).
- **Security-гейт на `PreToolUse` hook** (не `can_use_tool` — тот не видит Read/Glob/Bash):
  deny секретных и защищённых путей (casefold), web-egress закрыт; Bash разрешён по решению
  владельца, но режется в `docs_only`, по секретным токенам и абсолютному пути `journal_dir`.
  NO-EXFIL backstop на SPEAK остаётся страховкой текста.

### PWA / UI (v2 + v3 + проекты)

- **Свой клиент на `/client/`**: SPA `#/` ↔ `#/thread/<id>` — голос живёт при навигации
  (live-доказано). Vendored `pipecat.mjs`, голос-латч, светофор, статус-карточка Коры.
- **Иерархия проекты → треды**: сайдбар-дерево, активный проект (localStorage = правда),
  чип в композере; голос/HTTP-задачи идут в папку проекта через единый `_resolve_project_root`.
- **Реконнект без wall-clock** (iOS замораживает таймеры при локе): вотчдог поллит
  `GET /client/session-alive`, reload только когда сервер доступен и говорит «сессии нет».
  Приветствие «С возвращением…» под латчем, replay недоставленных critical.
- **Текстовый канал**: `AnthropicLLMClient` + tool_use-шейп, пер-тред история с холодной
  регидрацией из ленты (NO-EXFIL-регидрация), CSRF, `turn_lock`. С С4 текст под **тем же
  дневным денежным лимитом**, что и голос.
- **Удалённый доступ**: Tailscale serve (`https://temirlans-macbook-air.tail97957e.ts.net`),
  staging-канал `:8443` → 7861.

## 2. Кривое / неработающее

1. **Остаточная граница Фазы 0 — не sandbox.** С6 очищает окружение реального CLI через
   `execve`, безусловно запрещает мутацию `journal_dir` файловыми инструментами и не тратит
   дедлайн во время `AskUserQuestion`. Но Кора всё ещё локальный процесс: сетевой namespace не
   изолирован, а разрешённый Bash остаётся полным egress и может обходить лексический скан через
   относительные пути/обфускацию. Настоящая изоляция — долг Фазы 2.
2. **Live-DoD за Теро** (не автоматизируются): телефон с токеном — голос и чат; запрос без
   токена → 401; PWA-установка; hands-free (tap-to-connect под вопросом — autoConnect в
   бандле нет).
3. **С5-шероховатости** (ран-файл, не блокеры): протухший токен при живом звонке переоткрывает
   диалог каждые 5 с (поллинг `session-alive`); голос со старым токеном падает без диалога
   (самолечится соседним HTTP-поллом); `GET /client` без слэша → 401 вместо 307.
4. **Вечные жёлтые / запаркованные**: B15 (единственный xfailed), B35, B13-grounding, B31,
   mic-gesture §2.3.
5. **KV-1 голос-свитч живьём — NO-GO**: свитч голоса Fish = реконнект WS, +0.7 с при пороге
   500 мс. Сдана половина KV-1a (Play через REST). Не перестраивать, перепроверка стоит денег.
